

Vous trouverez ci-joints 3 fichiers:


- le fichier SQL Tables, qui crée la base, crée les tables, puis les remplie,

- le fichier SQL Requetes, qui contient toutes les requêtes demandées.


REMARQUE: Un programme JAVA crée par nos soins s'est chargé de remplir aléatoirement les tables.
De ce fait, certaines requêtes, bien que justes, fournissent un "Empty Set" comme résultat.
Nous aurions été forcés de remplir les tables manuellement si on souhaitait un résultat non vide pour chaque requête.
